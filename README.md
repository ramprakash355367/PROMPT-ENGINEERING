1. FundamentalsConcepts of Generative AI and Its Types
   
What is Generative AI?

Generative AI is a type of artificial intelligence that can create new content such as text, images, music, videos, or even code—rather than just analyzing or processing existing data.

What are Generative Models?

Generative Models are a class of models in Machine Learning that learn how data is structured and then generate new data that looks similar to the original dataset.

🔹 Types of Generative Models

       1.GANs (Generative Adversarial Networks)

           ○Two networks compete (generator vs discriminator)

           ○Produces highly realistic images

      2.VAEs (Variational Autoencoders)

          ○Encode data into a compressed form and then reconstruct it

      3.Transformers

          ○Used in text generation models like ChatGPT

3. What is an LLM and How is it Built?

What is an LLM 
LLM (Large Language Model) is an artificial intelligence model designed to understand and generate human language by learning from large amounts of text data.

How is it Built

    1. Data Collection
      ●Large amounts of text are collected (books, websites, articles)

      ●The dataset should be diverse and high-quality

    2. Data Preprocessing
       ●Clean the data (remove errors, duplicates)

       ●Convert text into tokens (small units like words or subwords)

    3. Model Architecture
        ●Built using transformer models

        ●Uses layers of neural networks to understand context and relationships in text

    4. Training the Model
         ●The model is trained to predict the next word in a sentence

         ●Uses powerful computers (GPUs/TPUs)

         ●Learns grammar, meaning, and patterns

    5. Fine-Tuning
         ●The model is further trained on specific tasks

         ●Improves accuracy and usefulness (e.g., chat, coding, translation)

    6. Evaluation & Testing
         ●Checked for performance, accuracy, and safety

         ●Errors are corrected and improvements are made

    7. Deployment
          ●The trained model is integrated into applications like chatbots, apps, and websites
